A Linguagem HTML

HTML (abreviação para a expressão inglesa HyperText Markup Language, que significa Lnguagem de Marcação de hipertexto) é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.

Alguns recursos da linguagem HTML vistos até este momento.

Recursos Tags Associadas Descrição
Links <a> Criar Links entre páginas
Listas <ul> e <li> Criar lista de itens não ordenados
<ol> e <li> Criar lista de itens ordenados
Imagens <img> Inserir Imagens
